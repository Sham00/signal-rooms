Data Methodology

Specialist clouds (Lambda, CoreWeave, RunPod): On-demand list prices from provider websites. Updated by fetch_data.py (Lambda Labs public API + manual cross-check for others) via GitHub Actions daily. Vast.ai: Median spot prices from marketplace; actual bids vary ±20%. Hyperscalers (AWS, GCP, Azure): On-demand list price per GPU, computed by dividing multi-GPU instance cost by GPU count. Hyperscaler prices rarely change — updated manually.

H100 Trend: Lambda Labs on-demand list price history compiled from public posts, pricing page snapshots, and community tracking. Not financial advice. Prices change without notice — always verify on provider websites before purchasing.
